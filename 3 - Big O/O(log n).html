Big O notation measures the relation between the amount of operations and the increase of inputs.

What is good code?

  • Readable
  • Scalable => Big O

Big O notation

O(1) - Constant time.
Runtime remains constant with each added input.
O(log n)
O(n) - Linear time.
Runtime increases linear with each added input.
O(n log n)
O(n^2)
O(2^n)
O(n!)
